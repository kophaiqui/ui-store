## Usage

```tsx
// Load dynamically in a component
const url = `https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;700&display=swap`;

// CSS font-family declaration
font-family: 'Playfair Display', serif;
```

All fonts support `font-display: swap` for performance. Weights are lazy-loaded on selection in the explorer.
